Open PDF Portable
====================

Run OpenPDF.exe directly from this folder.

No Python installation is required. OCR support for Danish and English is
included in the tesseract folder.

The application is intended for 64-bit Windows 10 or Windows 11.

Keep the entire folder together when copying it to a USB stick. Do not copy
only OpenPDF.exe because the _internal and tesseract folders are required.
